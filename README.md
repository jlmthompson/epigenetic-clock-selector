# Epigenetic Clock Selector ⏱

An interactive guide for researchers choosing which epigenetic clock to use.

**Live site:** https://jlmthompson.github.io/epigenetic-clock-selector/

## What it does
Enter your study context (tissue type, age range, study goal, array platform, disease context) and get ranked clock recommendations with:
- Plain-language rationale
- Key limitations
- PubMed references
- R package implementation
- Full comparison table of all clocks

## Clocks included
Horvath 2013, Hannum 2013, PhenoAge (Levine 2018), GrimAge (Lu 2019), DunedinPACE (Belsky 2022), PC Clocks (Higgins-Chen 2022), Knight 2016, Lee 2019, PedBE (McEwen 2020), Skin & Blood (Horvath 2018), EN Clock (Zhang 2019)

## Built for
Cardiovascular, paediatric, and neurodevelopmental genomics research.
